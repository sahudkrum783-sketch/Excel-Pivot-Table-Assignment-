# **Excel Pivot Table Assignment**

## Overview

This Excel assignment demonstrates the use of Pivot Tables to analyze a dataset containing sales transactions. The dataset includes columns such as Transaction ID, Salesperson, Region, Product, Category, Date, Revenue, and Cost.

All steps taken to complete the assignment have been documented in the attached **Word file**: [Excel PivotTable_Assignment_Steps taken.pdf](https://drive.google.com/file/d/1uVzEZAOx8uBuLvwvEnF8WJRpLaA4PF0H/view?usp=drive_link).

The assignment covers key Excel functionalities including:

* Creating Pivot Tables
* Filtering and sorting data
* Grouping dates by month
* Adding calculated fields
* Using slicers for interactive filtering
* Identifying best-selling products
* Applying conditional formatting
* Creating Pivot Charts
* Refreshing Pivot Tables with new data
